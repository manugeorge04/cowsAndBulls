# Cows and Bulls


## Prerequisites

- [NodeJS](https://nodejs.org/en/)


## Tech Stack Used

- [NodeJS](https://nodejs.org/en/)
- [ReactJS](https://reactjs.org/)
- [ExpressJS](https://expressjs.com/)


## Build and Run

### `npm install`

### `cd ./client && yarn install`

### `npm start`

### Open another terminal :

### `cd ./client && yarn run dev-server`
